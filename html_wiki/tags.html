HTML тэги

Тег Предназначение тега
<html> Тег содержащий внутрисебя всю html страницу.
<!DOCTYPE> Он обьявляет тип документа, также может указывать на версию языка разметки.
<!--...--> Используется для добавления комментариев.
<head> Тег с основными свойствами HTML-страницы.
<body> Тег, в котором находятся все отображаемые на странице элементы.
<title> Тег, для определения заголовка страницы.
<link> Одиночный тег, для подключения внешних таблиц стилей.
<h1> - <h6> Заголовки шести уровней (по убыванию размера).
<a> Создаёт гипертекстовые ссылки.
<img> Одиночный тег, который вставляет изображение на страницу с помощью аотрибута src,значением которого является адрес изображения.
<b> Тег,который даёт тексту жирное начертание.
<em> Тег,который делает текст курсивным.
<I> Тег,который делает текст курсивным.
<sub> Тег,при помощи которого можно написать нижний индекс.
<sup> Тег,при помощи которого можно написать верхний индекс.
<strong> Тег,который даёт тексту жирное начертание.
<pre> Тег,при использовании которого текст будет писать классицеским шрифтом,включая все пробелы.
<ul,li,ol> Тэги,при помоци которых создаётся список разных видов маркировки.
